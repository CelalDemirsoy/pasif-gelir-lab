# Pasif Gelir Lab

## Kurulum

### 1. Node.js yükle (yoksa)
https://nodejs.org adresinden indir ve yükle (v18 veya üzeri)

### 2. Terminali aç ve projeye gir
```bash
cd pasif-gelir-lab
```

### 3. Bağımlılıkları yükle
```bash
npm install
```

### 4. Geliştirme sunucusunu başlat
```bash
npm run dev
```

Tarayıcıda http://localhost:3000 adresine git.

## Build (Yayına alma)
```bash
npm run build
```
